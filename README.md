## Types
* Primitives
  * string
  ```javascript
  const foo = 'name';
  console.log(typeof foo); // string
  ```
  * number
  ```javascript
  const foo = 10;
  console.log(typeof foo); // number
  ```
  * boolean
  ```javascript
  let foo = true;
  console.log(typeof foo); // boolean
  ```
  * null
  * undefined
  * symbol

* Complex
  * object
  * array
  * function
